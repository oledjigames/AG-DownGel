Кроссплатформенное GUI-приложение для скачивания с GelBooru.com, построенное на PyQt6. Поиск и массовая загрузка изображений, GIF и видео по тегам с фильтрацией, исключением ИИ и организованным хранением.

## Возможности
- **Поиск по тегам**: Вводите теги через пробел (например, `catgirl solo`); поддержка синтаксиса GelBooru, как `rating:explicit`.
- **Фильтрация файлов**: Выберите изображения (.jpg, .png и т.д.), GIF или видео (.mp4, .webm); скачивает в подпапки (`images/`, `GIF/`, `videos/`).
- **Исключение ИИ**: Чекбокс добавляет `-ai-generated` для исключения ИИ-контента.
- **Поддержка API**: Опциональный User ID/API Key для повышения лимитов (получите на [GelBooru Account](https://gelbooru.com/index.php?page=account&s=options)).
- **Локализация**: Английский (по умолчанию), Русский, Китайский, Японский; расширяемо через `locales/*.json`.
- **Скачивания**: Сохраняются в `downloads/[очищенные_теги]/` для удобства.
- **Прогресс и ошибки**: Статус в реальном времени, неопределённый прогресс для пагинации.

## Установка
1. Убедитесь в Python 3.10+ и установите зависимости:
2. pip install PyQt6 requests
3. Запустите: `python main.py` (переименуйте скрипт при необходимости).
4. Папки создаются автоматически: `downloads/` (вывод) и `locales/` (переводы).

## Использование
1. Выберите язык из выпадающего списка (вверху).
2. Введите теги в поле поиска (например, `brazilian_miku hatsune_miku feet`).
3. Отметьте "Без ИИ" для исключения ИИ-генерированного.
4. Выберите типы файлов.
5. (Опционально) Введите User ID и API Key.
6. Нажмите "Скачать все" – файлы сохранятся в `downloads/[теги]/`.

## Примеры тегов
Из `guide.txt`:

**Kasane_teto -hatsune_miku rating:explicit**  
(скачивает teto без miku, с 18+ контентом)

**Опции рейтинга:**  
- `rating:explicit` (NSFW)  
- `rating:questionable` (пограничный)  
- `rating:sensitive` (взрослый)  
- `rating:general` (безопасный)

## Устранение неисправностей
- **0 постов?** Добавьте связанные теги (например, `hatsune_miku` для `brazilian_miku`); проверьте [DAPI docs](https://gelbooru.com/index.php?page=help&topic=dapi) GelBooru.
- **Лимиты скорости:** Используйте API Key для >100 результатов/страница.
- **Кастомные языки:** Добавьте `locales/[lang].json` с ключами как в `en.json`.
